 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Установить Git на свой ПК
Зарегистрироваться на Github
Создать свой SSH ключ
скопировать  URL репозитория SSH 
В нашем GitBash   использовать команду git clone и вставить нашу ссылку из github



Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

git init
git add .
git commit -m "My comments"
git remote 
git push -u



Как получить на локальный компьютер изменения с github.com

Проверить связан ли local репозиторий с Git
git remote -v
git pull



Какую информацию нам дают команда :
git status - показывает состояние файла в Git репоситории

git log - Список Коммитов которые на данный момент существуют в репоситории git 

git remote -v - (Fetch),(Push) - проверка - наш локальный репозиторий привязан к GitHub?
